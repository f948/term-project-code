    Part 2: Preparing your github repository
  9. Create an account on github at https://github.com/
  10. Login to your github account
  11. Create a repository called "chat application"
  12. Upload all the files you create in part 1 to that github repository

    13. Part 3: Hosting github repository code on cloud using heroku
  13. Create an account on heroku at https://www.heroku.com/
  14. Login to your heroku account
  15. Create a node.js buildpack
  16. Create a new app with whatever name you choose
  17. Navigate to the "deploy" section in your app dashboard
  18. Connect the github repository you created in part 2 with your new heroku app
  19. Click the button "Deploy Branch" to deploy your app to the cloud
  20. Click the button "Open App" to view your app
