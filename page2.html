How to install Node.js

  1. Go to the website https://nodejs.org/en/download/
  2. Click on the button that downlaods the node install that is appropriate for your operating system version
  3. Run the node installer when the downlaod finishes to install node.js on your device

How to install additional node modules

  1. Open up the command prompt on your operating system
  2. Navigate to the directory where node.js is installed
  3. Type "npm install module name" into the command prompt and the node modules will be installed
  4. Node: do not type npm install module name, intead make sure to replace module name with the actual module name.For example if I want to install express I would type in "npm install express" into the command prompt
  5. The node modules should then be installed and ready for use

How to run your first node.js application

  1. Copy and paste the following code into a text editor

    2. var http = require("http");
    http.createServer(function (request, response) {
    // Send the HTTP header
    // HTTP Status: 200 : OK
    // Content Type: text/plain
    response.writeHead(200, {'Content-Type': 'text/plain'});
    // Send the response body as "Hello World"
    response.end('Hello World\n');
    }).listen(3000);
  2. Save the file as app.js in the same directory as node.js is installed
  3. Open up the command prompt on your computer
  4. Navigate to the directory where app.js file was installed
  5. Type "node app.js" into the comamnd prompt
  6. Open up your web browser
  7. Type "http://localhost:3000/" into your web browser. A page should open up with the message "Hello world"

How to install React

  1. Open up the command prompt on your perating system
  2. Navigate othte same directory where node was installed
  3. Type "npm install -g create-react-app" into the prompt
  4. After the command is executed type "npx create-react-app react-app" into the prompt. This will create a new project named react-app
